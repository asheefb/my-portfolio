Freelance Hub - Project Details

Project Architecture

The application follows a microservices architecture with the following components:

  • API Gateway for routing and load balancing
  • Eureka Service Registry for service discovery
  • User Service for authentication and profile management
  • Job Service for job posting and application management
  • Messaging Service for communication between freelancers and clients
  • Notification Service for email and in-app notifications

Technical Implementation

  • Implemented JWT-based authentication with refresh token mechanism
  • Used MongoDB for storing user profiles, messages, and notifications
  • Implemented MySQL for structured data like jobs, applications, and reviews
  • Integrated AWS S3 for secure file uploads (portfolios, attachments)
  • Used RabbitMQ for asynchronous processing of notifications
  • Implemented real-time messaging using WebSockets
  • Created a responsive frontend using React.js and Bootstrap

Challenges & Solutions

  • Challenge: Service communication reliability
    Solution: Implemented circuit breaker pattern using Resilience4j
  • Challenge: Data consistency across services
    Solution: Used event-driven architecture with RabbitMQ
  • Challenge: Secure file uploads
    Solution: Implemented pre-signed URLs with AWS S3
